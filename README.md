# ArmaOBS

**Author:** *Shtufff*  
**Date:** 22 December 2025

This repository contains a `.bat` file. When configured properly, it will launch the game "Arma Reforger" and the recording/streaming software "OBS".

---

## Configuring the file to work with your system.

- Don't change lines 1 or 2.
- For line 4, change the path to your "OBS install location, specifically the folder named "64bit". Make sure the path is inside of the "".
- For line 6, copy the path from "Line 4" and point it to `obs64.exe`. Again inside the "".

---

## OBS Startup Parameters.

- `--startreplaybuffer` launches "OBS" with the Replay Buffer enabled, this is good for recording gameplay.

  A list of Startup Parameters can be found here:
  https://obsproject.com/kb/launch-parameters
